• About Consultant Communicator
    Consultant Communicator (ConsultComm) is a program that allows anyone managing multiple projects, clients or tasks to effectively keep track of exactly how long they've spent on each project.

  • Install
    Read the file install.html for step-by-step directions to install ConsultComm.

  • Starting Consultant Communicator
    If have installed ConsultComm using InstallAnywhere, just click on the icon that was created for you. If you have another distribution, simply double-click on the "CsltComm.jar" icon. Or, at the DOS prompt type:
    > javaw -jar CsltComm.jar (or java -jar CsltComm.jar on non-Windows platforms)

  • Running Consultant Communicator
    1. The Time Tracker
      Simply use the toolbar to use the following functions:
      • Start - Start the clock running.
      • Pause - Stop the clock from running.
      • Project - Maintain your projects by selecting:
        • Add Project - Place a new project at the end of the list
        • Delete Project - Deletes currently seleted project
        • Edit Project - Edit the currently selected project
        • Reset Timers - Reset ALL projects to have no elapsed time
      • Tools - Configuration and connectivity
        • Export to Table - Insert all of your projects and times into a JDBC database
        • JDBC Settings - Edit settings to connect to JDBC compliant databases and insert projects and times into tables. See Connecting to a database with ConsultComm for more details
        • Preferences - Change how ConsultComm looks and runs to suit your personal preferences
          • General - General options that allow you to specify how time is displayed, how animated icons are shown and how often to save your information to a file
          • Flags - What attributes a project can have.
            • Show Billable Flag - Allow the "Billable Project" checkbox to show up when editing or adding a project
            • Use Export to Database Flag - Allow the "Export Project to Database" flag. This allows you to mark individual projects to be sent or not to be sent to a database
          • Skins - Change the look & feel of ConsultComm. Note that GTK and KDE themes work in Windows, too.
            • Use Theme Pack - The location of a SkinLF theme pack. Themes are available at http://javootoo.l2fprod.com/plaf/skinlf/index.php
            • Use GTK Theme - Use a GTK designed theme
            • Use KDE Theme - Use a theme generated for KDE
          • Help - View the help documentation
      • Switching Projects
        When you are ready to switch to another project, simply click on its name on the list. ConsultComm automatically saves the old project's elapsed time and watches elapsed time on the new project.
      • Sorting Projects
        Just click on the title of a column to sort it in ascending order. Click it again to sort in decending order. New projects are automatically sorted into the list.
      • Total vs. Billable Time
        To see your billable (instead of total) time, simply click on the "Total:" box at the bottom of ConsultComm. When you want to switch back, simply click on the "Billable:" box.
      • Saving Preferances
        Projects, time and preferances are automatically saved at specified increments and when you exit Consultant Communicator.
